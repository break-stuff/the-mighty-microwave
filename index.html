Welcome to the Mighty Microwave!

Feel free to use the key pad to type in the desired time and click Start to start the microwave or use one of the Presets to quickly handle those repetitive or hard-to-remember tasks.

Setting Presets

To program a preset all you have to do is:

  1. Push the Set Presets switch at the bottom of the microwave to put the microwave in "program" mode
  2. Push the button you wish to program
  3. Fill out the form with the name you wish to display on the button and the preset time
  4. Push the Set Presets switch again to resume normal functionality
